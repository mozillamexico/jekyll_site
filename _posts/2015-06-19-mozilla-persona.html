Una forma más fácil de identificarte

Mozilla Persona: es un sistema de autenticación para la web completamente descentralizado y seguro basado en el protocolo abierto de BrowserID.

https://www.youtube.com/watch?v=azwP0eHML00

Mozilla Persona sustituye el uso de múltiples contraseñas

En lugar de tener que inventar para cada sitio un nombre de usuario y una contraseña, que en ocasiones es dificil de recordar; Mozilla Persona te permite usar tu dirección de correo para iniciar sesión en cualquier sitio web que lo tenga implementado.

Sitios como Mozilla Webmaker, Trovebox y Voost utilizan Mozilla Persona en lugar de nombres de usuario para identificarte.

 Mozilla Persona es flexible

 

En Mozilla Persona tu identidad es tu dirección de correo electrónico.

Puedes usar tantas direcciones como desees, pero aun así solo necesitas una contraseña.

 

 

¿Qué ventajas tiene Mozilla Persona?

  • Puedes usar unas credenciales fáciles de usar para todos los sitios que tengan implementado Mozilla Persona.
  • No tienes que confiar tu contraseña a un sitio web. Además, no te podrán robar la contraseña si hackean uno de esos sitios.
  • Puedes utilizar Persona con tantas direcciones de mail como quieras. Esta opción te permite mantener distintos perfiles digitales (trabajo, colegio, casa, etc.).
  • Mozilla Persona no le permite a tu proveedor de correo electrónico rastrear las páginas web que visitas (Política de Privacidad de Persona), una vez que te conectas a un sitio web, crea un muro entre tu identificación y tu actividad. El historial de los sitios que visitas se guarda únicamente en tu equipo.

Conéctate con Mozilla Persona, la manera más fácil y segura de identificarse.
